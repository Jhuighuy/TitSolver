# User Manual

## Documentation map

- [Theory Guide](./theory.html) outlines the mathematical framework and
  connects the notation used in the manual to the implementation choices in the
  solver.
- [SPH Framework](./theory/sph.html) walks through kernel interpolation,
  density estimation, and first-order differential operators used in SPH.

## How to use this manual

Use the links on this page to jump into the theory material, the breadcrumbs to
move back up a level, and the page outline at the bottom to revisit headings
within the current topic.

The current manual is intentionally compact. As more guides are added, this
entry page will stay stable as the main landing page for the GUI help system
and packaged documentation.
