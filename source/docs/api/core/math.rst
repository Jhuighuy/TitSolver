Math routines
=============

.. code-block:: cpp

   #include <tit/core/math.hpp>

Overview
--------

The header file ``tit/core/math.hpp`` comprises a collection of mathematical
routines.
These routines predominantly consist of overrides for standard
mathematical functions designed for real-numbered inputs, occasionally
complemented by overrides suitable for integer inputs.
